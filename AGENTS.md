# Available Skills

## Invocation
Use `load_skill` with the skill name to invoke.

## Frontend Design (`frontend-design`)
Design and implement frontend pages, mockups, and UI components.
- **Use for:** Creating new pages, redesigning existing UI, generating mockups
- **Example:** "Design a product detail page mockup"

## Code Review (`zen-review`)
Expert code review for correctness, security, performance, and quality.
- **Use for:** Reviewing changes before committing, security audits, performance reviews
- **Example:** "Review the auth system for security issues"

## Planning (`plan`)
Break down complex tasks into step-by-step implementation plans.
- **Use for:** Large features, refactoring, architecture changes
- **Example:** "Plan the checkout flow implementation"

## Research (`research`)
Fast codebase exploration to find files, patterns, and understand architecture.
- **Use for:** Understanding unfamiliar parts of the codebase
- **Example:** "Find all admin route files"

## Browser Automation (`agent-browser`)
Automate browser interactions for testing and QA.
- **Use for:** Testing forms, navigation flows, visual verification
- **Example:** "Test the login flow end-to-end"
